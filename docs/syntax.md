# Zeryon Syntax v0.1.0

## Palabras clave reservadas

- `app` → define la aplicación principal
- `page` → define una pantalla
- `layout` → contenedor visual (Column, Row, Stack, etc.)
- `component` → define un componente reutilizable
- `import` → importa módulos
- `start` → página inicial de la app
- `navigate` → cambio de pantalla
- `fetch` → llamada a API
- `state` → variable reactiva
- `style` → define estilos visuales
- `font` → tipografía
- `color` → colores hex o rgba
- `asset` → imágenes, videos, audio
- `export` → exportar componentes o funciones

## Reglas de sintaxis

- Todo archivo `.zy` usa **inglés** excepto comentarios.
- Comentarios: `//` para línea, `/* */` para bloque
- Identación: 4 espacios (no tabs)
- Nombres de variables: camelCase
- Nombres de componentes: PascalCase
- Strings: dobles comillas `"texto"`
- Números: enteros y decimales estándar
- Listas: `[item1, item2]`
- Objetos: `{ key: value }`
